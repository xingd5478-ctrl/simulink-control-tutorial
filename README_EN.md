## Getting Started

Each tutorial is a `.m` script. Run in the MATLAB command window:

```matlab
>> t01_signal_basics       % Basics
>> t10_state_feedback      % LQR control
>> t15_code_generation     % Generate C code

>> run_all_tutorials       % Run all 15 lessons; figures saved to docs/images/
```

Scripts automatically create Simulink models (`.slx`), configure blocks, run simulations, and generate result plots.

**Requires:** Control System Toolbox (`place`, `lqr`, `lqe`, `ss2tf`, `c2d`, etc.)

## Learning Path

```
t01-t08 Fundamentals ──→ t09 State-Space ──→ t10 LQR Control
                                               │
                  t13 DC Motor ←── t11 Observer ←── t12 Kalman
                       │
                  t14 PMSM/FOC ←── t15 Code Gen → Embedded Deployment
```

After all 15 lessons, you will be able to:
- Derive state-space models from physical laws
- Design LQR/Kalman optimal controllers and observers
- Go from continuous → discrete → C code, deploy to STM32
- Understand cascaded PI control and FOC vector control in industrial practice
