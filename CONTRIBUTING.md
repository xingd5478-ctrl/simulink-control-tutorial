# Contributing

Issues and PRs are welcome.

## Reporting Issues

- Use a descriptive title
- Include MATLAB version and OS
- For bugs, include the error message and steps to reproduce

## Pull Requests

1. Open an issue first to discuss what you'd like to change
2. Keep changes focused — one feature or fix per PR
3. Follow the existing coding style (Chinese comments in `.m` files, clear section headers)

## Ideas for Contribution

- New tutorial topics (robust control, MPC, adaptive control, etc.)
- Translations (Japanese, Korean, etc.)
- Bug fixes or compatibility patches for older MATLAB versions
- Improved model visualization or diagram exports
