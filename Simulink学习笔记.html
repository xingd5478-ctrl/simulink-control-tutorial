Phase 1 · Simulink 基础（t01-t08）
🔌

t01 · 信号基础

t01_signal_basics

认识 Simulink 三大基本要素：模块 Block、信号线 Signal、仿真运行 Simulation。搭建第一个信号流：Sine Wave → Gain → Scope。

add_block('simulink/Sources/Sine Wave', ...) % 拖模块 add_line(mdl, 'Sine Wave/1', 'Gain/1') % 连线 set_param(..., 'Amplitude', '3') % 改参数 sim(mdl) % 运行
💡 模块 = 乐高积木，连线 = 流水管，仿真 = 打开水龙头。
🧮

t02 · 数学运算与信号路由

t02_math_blocks

Sum 加减、Product 乘除、Mux/Demux 信号捆绑与拆分。两列正弦波相加 = 叠加原理（傅里叶思想）。

☕

t03 · 一阶系统与传递函数

t03_first_order_sys

理解系统动态特性：G(s) = K/(τs+1)。K = 稳态增益，τ = 时间常数（惯性）。比较 τ=0.5/2/5 对阶跃的响应。

🫖 一杯热水降温——水温不会瞬间降到室温，而是慢慢逼近。τ 越小 = 杯子越小散热越快。
🎢

t04 · 二阶系统与阻尼特性

t04_second_order_sys

G(s) = ωn²/(s² + 2ζωns + ωn²)。ζ 阻尼比决定振荡行为：ζ=0 纯振荡、0<ζ<1 衰减振荡、ζ=1 临界阻尼、ζ>1 过阻尼。

🔗 ζ 太小 → 超调严重 → 需要 PID 的 D 参数来 "增加阻尼"，这就是 t04 是 t05 前置的原因。
🎯

t05 · PID 反馈控制

t05_pid_control

经典控制核心：u(t) = Kp·e + Ki·∫e + Kd·de/dt。P 有误差立刻反应、I 累积消除残余、D 预测抑制超调。分别验证 P/I/D 单独作用 + 联合闭环调参。

📂

t06 · 信号导入导出

t06_sources_and_sinks

打通 MATLAB ↔ Simulink 数据交换：From Workspace (进口)、To Workspace/To File (出口)、Display/Scope (监视)。学完这课就能把仿真结果拿到 MATLAB 做 FFT、统计等后处理。

📦

t07 · 子系统封装

t07_subsystem

将复杂模型折叠成黑盒子——理解层次化建模思想。把 PID+Plant 整个闭环压成一个 Subsystem，输入=设定值，输出=实际值。

🎛️

t08 · Mask 参数化封装

t08_masking

自定义参数对话框，让子系统可复用、可配置。双击弹出参数窗口，输入 Kp/Ki/Kd 即可——不用每次打开子系统内部。

Phase 2 · 控制理论（t09-t12 + t16-t21）
📐

t09 · 状态空间模型

t09_state_space

现代控制理论基础：ẋ=Ax+Bu, y=Cx+Du。从传递函数到状态空间，理解"状态"的物理意义，能控性/能观性。

🎯

t10 · LQR 最优控制

t10_state_feedback

极点配置 → LQR → Riccati 方程。Q/R 权重矩阵调参策略：Q 大=重视性能，R 大=节省能量。多变量系统设计的核心工具。

👁️

t11 · Luenberger 状态观测器

t11_observer

当状态不可直接测量时——用观测器"估计"状态。x̂̇ = Ax̂ + Bu + L(y-ŷ)。对偶原理，分离原理验证。

🔮

t12 · Kalman 滤波

t12_kalman_filter

噪声环境下的最优状态估计。过程噪声 Q 与测量噪声 R 建模，lqe() 设计。与 Luenberger 对比：Kalman 在噪声下更优。

📈

t16 · 频域分析

t16_freq_domain

Bode 图 / Nyquist 图 / Nichols 图。增益裕度 GM、相位裕度 PM——频域稳定性判据。不同增益 K 对稳定裕度的影响。

✏️

t17 · 超前-滞后校正

t17_lead_lag

根轨迹法设计 Lead/Lag 补偿器。Lead = 增加相位裕度（改善暂态），Lag = 提高低频增益（改善稳态）。

🔬

t18 · 系统辨识

t18_system_id

从实验数据到数学模型。阶跃响应法提取 K/τ/θ，最小二乘法拟合。模型验证：残差分析、交叉验证。

🛡️

t19 · H∞ 鲁棒控制

t19_robust_control

主动考虑参数不确定性（±20%）。混合灵敏度 S/KS/T 整形 → hinfsyn 求解 → μ 分析。LQR 单一名义模型 vs H∞ 全局鲁棒。

H∞ 设计流程： ureal('m',1.0,'±20%') → W1/W2/W3 加权 → hinfsyn(P,1,1) → γ<1 ? 成功 : 放松权重
🧠

t20 · 模型预测控制 MPC

t20_mpc

滚动优化 + QP 约束求解 + 显式 MPC。带输入/状态约束的多变量控制——工业过程控制的标配。

⚡

t21 · 滑模控制 SMC

t21_sliding_mode

非线性鲁棒控制。滑模面 s=λe+ė → 0，等效控制 + 切换控制。抖振抑制：饱和函数 / Super-Twisting 算法。

Phase 3 · 机电系统 + 部署（t13-t15）
⚙️

t13 · DC 电机控制

t13_dc_motor

电磁转矩 + 机械负载耦合建模。级联 PI（电流环+转速环），LQR 对比。执行器建模实战。

🔋

t14 · PMSM + FOC 矢量控制

t14_foc_pmsm

Clarke → Park 坐标变换，d-q 解耦，SVPWM 原理。无刷电机矢量控制全流程——新能源车/机器人关节的核心技术。

💻

t15 · C 代码生成与嵌入式部署

t15_code_generation

连续→离散 (c2d)，离散 LQR (dlqr)，生成 C 代码，FreeRTOS 集成框架。不依赖 Simulink Coder——手动完成全流程。

连续设计 → c2d 离散化 → dlqr 离散增益 → C 代码 → STM32
🗺️
